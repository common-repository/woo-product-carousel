WooCommerce Product Carousel is a simple and fully Responsive Carousel plugin developed for WordPress which comes with lots of useful stunning features. It allows you to create unlimited Product Carousel on your website anywhere you like easily via shortcode and easy admin setting. By using WooCommerce Product Carousel, you can display more products at the same time and let them scroll or navigate automatically so that your visitors can see them all and increase sales of your shop.

Embed simply in anywhere post/page/widget using shortcode 
<code>[woo-product-carousel]</code>

= How to Use WooCommerce Product Carousel =

Installing this plugin as regular WordPress plugin.

After install, you are ready to use Product Slider in your post, page, widget or anywhere you like. How to create Product Slider? lets see.


1. Upload the folder woo-product-carousel to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
3. Call shortcode to your WordPress editor directly, by using 

`
[woo-product-carousel]
`
Call shortcode to your php template file using 
`
<?php  echo do_shortcode('[woo-product-carousel]'); ?>
`

Shortcode attributes
`
theme 		= theme_one, theme_two, theme_three
dots 		= true / false
nav 		= true / false
auto_play 	= true / false
img_border 	= true / false
items 		= use any number
`
